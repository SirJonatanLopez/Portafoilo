Festival de Música

Web sobre un evento de música, con información básica del evento como bandas, fechas y precios. Se ha desarrollado utilizando varias herramientas de Gulp como: 'SASS' para los estilos, 'imagemin' para optimizar el tamaño de las imágenes, así como 'cssnano' y 'tercer' para optimizar los archivos CSS y JS respectivamente

Tienda FrontEnd

Plantilla de una tienda virtual, en este caso de camisetas. Se compone de la página principal donde se enseñan los productos; a modo de ejemplo la página de información y compra de uno de los productos, y finalmente una parte de información sobre la tienda. En este diseño en particular todas las distribuciones se manejan con Grid y en la hoja de CSS se maneja la metodología BEM.
