Bienes Raíces 2.0

Website de Bienes Raíces completamente dinámico. Se compone de una parte accesible para el público donde se muestran diferentes propiedades en venta, las cuales se cargan desde una base de datos MySQL mediante PHP en su forma orientada a objetos apoyándose en Composer para cargar los módulos necesarios; una sección con entradas de blog también es cargada desde la base de datos. Por otro lado, se tiene la parte administrativa a la cual se accede después de iniciar sesión en la sección de login, desde esta sección se accede a un listado donde se pueden agregar, actualizar y borrar tanto las propiedades como los vendedores modificando la respectiva base de datos. También cabe mencionar que en la sección de contacto se pueden enviar emails a mi inbox del servidor Mailtrap.

En cuanto a la construcción del sitio, se utilizó arquitectura MVC para mantener un mejor orden y mejor control de los links para las diferentes secciones de la página.

App Salón de Belleza

Diseño de página web de un salón de belleza que se puede adaptar a cualquier tipo de negocio. Se compone de tres secciones: una para seleccionar los servicios deseados, otra para ingresar los datos del cliente y finalmente un resumen. Se utiliza un archivo Json para la base de datos de los servicios y JS para desplegar cada sección y validar los datos ingresados por el usuario.
