App de Héroes

App con una base de datos de diferentes superheroes cargados desde un sencillo servidor JSON. Desde un menú se puede acceder a la lista actual, agregar o modificar los heroes existentes. Se utiliza LazyLoad para cargar las rutas hijas de cada sección, Angular Material para los estilos, y un Guard para tener que hacer login antes de acceder al listado de Heroes.

App Buscador de Paises

App elaborada en Angular que permite obtener información básica de un país (población, bandera, código numérico, etc) realizando una búsqueda por su nombre, capital o región a la que pertenece utilizando como base la API de REST Countries haciendo las respectivas request con el módulo de HttpClient. Está integrada por un módulo de routing para el manejo de rutas, un componente para el sidebar de navegación, una interface para indicar las propiedades de cada país, un servicio donde se hacen las request y los componentes individuales de cada sección.

App Buscador de Gifs

Aplicación elaborada mediante Angular que permite buscar gifs ingresando la palabra deseada. Se compone de una barra lateral que muestra las últimas diez búsquedas y permite volver a mostrar los gifs relacionados, y una sección donde se muestran los gifs encontrados con ayuda de la API de Giphy. Su estructura está conformada por un módulo general, los componentes de: búsqueda, sidebar, muestra de resultados, y un archivo de servicios con toda la lógica de TS y donde se hace el request al servidor de Giphy con el módulo de HttpClient.

Bienes Raíces 2.0

Website de Bienes Raíces completamente dinámico. Se compone de una parte accesible para el público donde se muestran diferentes propiedades en venta, las cuales se cargan desde una base de datos MySQL mediante PHP en su forma orientada a objetos apoyándose en Composer para cargar los módulos necesarios; una sección con entradas de blog también es cargada desde la base de datos. Por otro lado, se tiene la parte administrativa a la cual se accede después de iniciar sesión en la sección de login, desde esta sección se accede a un listado donde se pueden agregar, actualizar y borrar tanto las propiedades como los vendedores modificando la respectiva base de datos. También cabe mencionar que en la sección de contacto se pueden enviar emails a mi inbox del servidor Mailtrap.

En cuanto a la construcción del sitio, se utilizó arquitectura MVC para mantener un mejor orden y mejor control de los links para las diferentes secciones de la página.

App Salón de Belleza

Diseño de página web de un salón de belleza que se puede adaptar a cualquier tipo de negocio. Se compone de tres secciones: una para seleccionar los servicios deseados, otra para ingresar los datos del cliente y finalmente un resumen. Se utiliza un archivo Json para la base de datos de los servicios y JS para desplegar cada sección y validar los datos ingresados por el usuario.
